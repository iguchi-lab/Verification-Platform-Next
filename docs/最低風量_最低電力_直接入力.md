## Part 1: 関連する入力項目

下記は 暖房・冷房 別に設定されます。

### 1.1 入力一覧

| 入力名 | 入力型 | 入力値 | 内部値 | 既定値 | 変数名 |
|-------|--------|--------|-------|-------|--------|
| ファン消費電力から換気分を引くか | 選択 | 「換気分を引く」 / 「換気分を引かない」 | `1` / `2` | `1` | `subtract_ventilation_power` |
| 最低風量（入力しない or 入力する） | 選択 | 「入力しない」 / 「入力する」 | `1` / `2` | `1` | `input_V_hs_min` |
| 最低風量 [m³/h] | 数値 | 正の実数 [m³/h] | — | — | `V_hs_min` |
| 最低電力（入力しない or 入力する） | 選択 | 「入力しない」 / 「入力する」 | `1` / `2` | `1` | `input_E_E_fan_min` |
| ファン消費電力算定方法 | 選択 | 「直線近似法」 / 「風量三乗近似法」 | `1` / `2` | — | `E_E_fan_logic` |
| 最低電力 [W] | 数値 | 正の実数 [W] | — | — | `E_E_fan_min` |

**ファン消費電力算定方式の種類**

| 点 | 風量 x | 消費電力 y（直線近似法） | 消費電力 y（風量3乗近似法） |
|----|--------|--------------------------|-------------------------------|
| 最低点（直接入力） | `V_hs_min` | `E_E_fan_min` | `E_E_fan_min` |
| 算定点 | `V_hs_supply` | `E = a · V + b` | `E = a · V³ + b` |
| 定格点 | `V_hs_dsgn` | `P_fan_rtd` | `P_fan_rtd` |

### 2.2 ミクロ視点: ステップ2 の変更前後

ステップ2 (`calc_Q_UT_A`) では、各区画の換気風量 `V_vent_g_i` を集計して `V_hs_vent` を決定する。
最低風量 直接入力:あり の場合、 `V_vent_g_i` の合計が `V_hs_min` がとなるように `V_vent_g_i` が決定する。

#### 2.2.1 Before（従来式）

```mermaid
flowchart LR

Vg["標準住戸の全般換気量と室面積から算出<br/>V_vent_g_i"]

Vg --> OutSum(["V_hs_vent = Σ V_vent_g_i"])

style Vg fill:#e1ffe1
style OutSum fill:#e1ffe1
```

---

#### 2.2.2 After: 最低風量 直接入力: あり

```mermaid
flowchart LR

Vmin["V_hs_min ★<br/>最低風量 直接入力"]
Vg["従来 V_vent_g_i"]

Vg --> Calc["V_vent_g_i = 従来V_vent_g_i × V_hs_min / Σ V_vent_g_i"]
Vmin --> Calc
Calc --> OutSum(["V_hs_vent = Σ V_vent_g_i"])

style Vg fill:#e1ffe1
style Vmin fill:#ffe1cc
style Calc fill:#ffe1cc
style OutSum fill:#e1ffe1
```

> V_hs_vent はこの値のまま ステップ4 へ渡される（ステップ4 では下限の再適用は行わない）。

#### 2.3.2 After: 換気分を引かない & 最低風量なし

**最低風量・最低電力の直接入力なしで、「換気分を引かない」を選択した場合。**

従来式(2.3.1)との違い:
- 換気分のファン消費電力 $$P_{fan,vent} = f_{SFP} \times V_{hs,vent}$$ を差し引かず、消費電力に含める
- サーモOFF（$$q_{hs} \ge 0$$）でも電力を 0 にせず、**暖冷房期間中は常にファン消費電力を計上**する
  - 暖房: 暖房期間のみ計上
  - 冷房: 冷房期間 + 中間期に計上

```mermaid
flowchart LR

subgraph 入力
    V(["V_hs_supply<br/>給気風量"])
    V_dsgn(["V_hs_dsgn<br/>設計風量"])
    V_vent(["V_hs_vent<br/>換気風量<br/>下限考慮なし"])
    P(["P_fan_rtd<br/>定格消費電力"])
end

V --> Ratio["換気補正風量比<br/>(V_supply - V_vent) / (V_dsgn - V_vent)"]
V_dsgn --> Ratio
V_vent --> Ratio

Ratio --> Power["消費電力計算<br/>(P_fan_rtd - f_SFP・V_vent) × ratio + f_SFP・V_vent"]
P --> Power
V_vent --> Power

Power --> Check{暖房期？（暖房計算時）<br/>冷房期または中間期？（冷房計算時）}
Check -->|Yes| Result(["E_E_fan"])
Check -->|No| Zero([0])

style Ratio fill:#e1ffe1
style Power fill:#ffe1cc
style Check fill:#ffffcc
```

> **ポイント**: 従来式では `P_fan_rtd` から換気分 `f_SFP・V_vent` を差し引いた上で風量比を掛けていたが、「換気分を引かない」ではその差し引いた分をそのまま加算して戻す。結果として、ファンの全消費電力（空調分 + 換気分）が暖冷房設備の消費電力として計上される。

#### 2.3.3 After: 最低風量 直接入力: あり & 最低電力 直接入力: なし

**最低風量のみ指定。計算ロジックは Before と同じだが、V_hs_vent が Step 2 で既に上書き済み。従来式とは異なり、サーモOFF時も電力を計算する（0にならない）。**

> **注意**: V_hs_vent の修正は Step 2 (calc_Q_UT_A) で実施済み（max(V_hs_min, Σ V_vent_g_i) または V_hs_min）

```mermaid
flowchart LR

subgraph 入力
    V([V_hs_supply<br/>給気風量])
    V_dsgn([V_hs_dsgn<br/>設計風量])
    V_vent([V_hs_vent ★<br/>換気風量<br/>下限考慮あり])
    P([P_fan_rtd<br/>定格消費電力])
end

V --> Ratio[換気補正風量比<br/>V_supply-V_vent ÷ V_dsgn-V_vent ]
V_dsgn --> Ratio
V_vent --> Ratio

Ratio --> Power[消費電力計算<br/>（P_fan_rtd - f_SFP・V_vent）・ratio]
P --> Power
V_vent --> Power

Power --> Check{暖房期？（暖房計算時）<br/>冷房期または中間期？（冷房計算時）}
Check -->|Yes| Result([E_E_fan])
Check -->|No| Zero([0])

style Ratio fill:#e1ffe1
style Power fill:#e1ffe1
style Check fill:#ffffcc
style V_vent fill:#ffe1cc
```

#### 2.3.4 After: 最低風量 直接入力: あり & 最低電力 直接入力: あり

**最低風量 + 最低電力を指定。カスタム消費電力式を使用。従来式とは異なり、サーモOFF時も電力を計算する（0にならない）。**

> **注意**: V_hs_vent の修正は Step 2 (calc_Q_UT_A) で実施済み（max(V_hs_min, Σ V_vent_g_i) または V_hs_min）

```mermaid
flowchart LR

subgraph 入力パラメータ
    V([V_hs_supply<br/>給気風量])
    V_vent([V_hs_vent ★<br/>換気風量<br/>下限考慮あり])
    V_dsgn([V_hs_dsgn<br/>設計風量])
    P([P_fan_rtd<br/>定格消費電力])
end

subgraph 追加入力パラメータ
    E_min([E_E_fan_min ★<br/>最低電力 直接入力])
    Logic([E_E_fan_logic ★<br/>算定方式])
end

V --> FlowDecision{サーモOFF?<br/>q_hs ≦ 0}
V_vent --> FlowDecision

FlowDecision -->|OFF| UseMin["使用風量 = V_hs_vent"]
FlowDecision -->|ON| UseActual["使用風量 = max(V_hs_supply, V_hs_vent)"]

Logic --> FormulaSelect{算定方式選択<br/>E_E_fan_logic}

FormulaSelect -->|直線近似法| LinearFormula["直線近似式<br/>E = a×V + b"]
FormulaSelect -->|風量三乗近似法| CubicFormula["風量3乗近似式<br/>E = a×V³ + b"]

LinearFormula --> PowerCalc["係数計算 & 電力算出<br/>2点: (V_hs_vent, E_min), (V_dsgn, P_rtd)"]
CubicFormula --> PowerCalc

UseMin --> PowerCalc
UseActual --> PowerCalc
V_vent --> PowerCalc
E_min --> PowerCalc
P --> PowerCalc
V_dsgn --> PowerCalc

PowerCalc --> Check{暖房期？（暖房計算時）<br/>冷房期または中間期？（冷房計算時）}
Check -->|Yes| Result([E_E_fan])
Check -->|No| Zero([0])

style FlowDecision fill:#ffffcc
style FormulaSelect fill:#ffe1cc
style PowerCalc fill:#ffe1cc
style V_vent fill:#ffe1cc
style E_min fill:#ffe1cc
style Logic fill:#ffe1cc
```
