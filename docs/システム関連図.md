# システム関連図

## 概要
本システムは、建築物の省エネルギー性能を計算・検証するためのプラットフォームです。
建築研究所が提供するオープンソースのpyheesをベースに、IBECsの全館暖冷房委員会が計算ロジックの検証用に開発・運用しています。

## モデル図

