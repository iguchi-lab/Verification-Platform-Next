# ステークホルダーマップ

## 概要
本システムは、建築物の省エネルギー性能を計算・検証するためのプラットフォームです。
建築研究所が提供するオープンソースのpyheesをベースに、IBECsの全館暖冷房委員会が検証版として開発・運用しています。

## モデル図

    System(platform, "検証プラットフォーム", "pyhees検証版を用いた検証作業を行うプラットフォーム")
    System(pyheesTest, "pyhees（検証版）", "建築物の省エネルギー性能計算プログラム（検証用カスタマイズ版）")
