# プロセスモデル：消費電力計算（標準）

## 概要

本ドキュメントでは、暖房・冷房消費電力の標準計算プロセスを詳細に記載します。

暖房と冷房は同じプロセスで処理されるため、統合して記述します。

**記法:** 
- `{H|C}` : 暖房はH、冷房はC
- `{温度|温度・湿度}` : 暖房は温度のみ、冷房は温度・湿度の両方

**注意:** 負荷計算（calc_heating_load/calc_cooling_load）は`pyhees`既存ライブラリをそのまま使用しています。

**各モデルの特徴:**

| type | 名称 | 特徴 |
|------|------|------|
| type1 | ダクト式セントラル | 機器仕様による性能曲線 |
| type2 | RAC現行省エネ法 | JIS C 9612基準のCOP |
| type3 | RAC潜熱評価 | 潜熱負荷を詳細評価 |
| type4 | 電中研モデル | R係数による特性曲線 |

---

## 消費電力計算の全体フロー

```mermaid
flowchart TD
    Start[消費電力計算開始] --> Step1[ステップ1<br/>送風機設計風量の決定<br/>V_hs_dsgn_H or V_hs_dsgn_C]
    
    Step1 --> Step2[ステップ2<br/>未処理負荷と中間変数の計算<br/>calc_Q_UT_A 呼び出し]
    
    Step2 --> Step3[ステップ3<br/>熱源機平均能力の計算<br/>q_hs_H_d_t or q_hs_CS_d_t, q_hs_CL_d_t]
    
    Step3 --> Step4[ステップ4<br/>送風機消費電力の計算<br/>E_E_fan_d_t]
    
    Step4 --> Step5[ステップ5<br/>消費電力の計算<br/>E_E_d_t]
    
    Step5 --> End([E_E_H_d_t or E_E_C_d_t])
```

### 消費電力計算の主要変数

| 変数名 | 単位 | 説明 | 次元 |
|--------|------|------|------|
| `V_hs_dsgn_{H\|C}` | m³/h | 送風機設計風量 | スカラー |
| `Q_UT_H_d_t_i` | MJ/h | 未処理暖房負荷 | 5区画×8760時間 |
| `E_C_UT_d_t` | MJ/h | 未処理冷房負荷（一次エネ換算済） | 8760時間 |
| `q_hs_H_d_t` | W | 熱源機平均暖房能力 | 8760時間 |
| `q_hs_CS_d_t` | W | 熱源機平均冷房顕熱能力 | 8760時間 |
| `q_hs_CL_d_t` | W | 熱源機平均冷房潜熱能力 | 8760時間 |
| `E_E_fan_{H\|C}_d_t` | kWh/h | 送風機消費電力 | 8760時間 |
| `E_E_{H\|C}_d_t` | kWh/h | 消費電力 | 8760時間 |

## ステップ1: 送風機設計風量の決定

```mermaid
flowchart TD
    Start[設計風量決定] --> Input{直接入力<br/>V_hs_dsgn > 0?}
    
    Input -->|Yes| Direct[入力値を使用<br/>V_hs_dsgn_H or V_hs_dsgn_C]
    
    Input -->|No| Type{計算モデル判定}
    
    Type -->|type1: ダクト式<br/>type3: RAC潜熱| UseInput[入力値から計算<br/>V_fan_rtd → V_fan_dsgn]
    
    Type -->|type2: RAC現行<br/>type4: 電中研| UseSpec[定格能力から計算<br/>q_rtd → V_fan_rtd → V_fan_dsgn]
    
    Direct --> Result[V_hs_dsgn 決定]
    UseInput --> Result
    UseSpec --> Result
    
    style Input fill:#ffffcc
    style Type fill:#ffffcc
```

**計算式:**
- `V_fan_dsgn = V_fan_rtd × 係数`（詳細は仕様書参照）

## ステップ2: 未処理負荷と中間変数の計算（calc_Q_UT_A 呼び出し）

このステップでは `calc_Q_UT_A()` を呼び出します。

**取得する変数:**

| 変数名 | 暖房時 | 冷房時 | 説明 |
|--------|--------|--------|------|
| 未処理負荷 | `Q_UT_H_d_t_i` | `E_C_UT_d_t` | 機器で処理できない負荷 |
| 吹出温度 | `Theta_hs_out_d_t` | `Theta_hs_out_d_t` | 熱源機からの吹出温度 |
| 吸込温度 | `Theta_hs_in_d_t` | `Theta_hs_in_d_t` | 熱源機への吸込温度 |
| 吹出湿度 | - | `X_hs_out_d_t` | 熱源機からの吹出絶対湿度 |
| 吸込湿度 | - | `X_hs_in_d_t` | 熱源機への吸込絶対湿度 |
| 給気風量 | `V_hs_supply_d_t` | `V_hs_supply_d_t` | 各区画への給気風量合計 |
| 換気風量 | `V_hs_vent_d_t` | `V_hs_vent_d_t` | 換気のための風量 |
| その他 | `Theta_ex_d_t`, `C_df_H_d_t` | `Theta_ex_d_t` | 外気温度、デフロスト補正係数（暖房のみ） |

### calc_Q_UT_A の全体フロー

```mermaid
flowchart TD
    Start([calc_Q_UT_A 開始]) --> A[2-A. 初期化・気象データ読込]
    
    A --> B[2-B. 床下温度の計算]
    
    B --> C[2-C. 各区画の設定温度・湿度の決定]
    
    C --> D[2-D. 各区画の供給風量の計算]
    
    D --> E[2-E. 給気温度・湿度の計算]
    
    E --> F[2-F. 吹出温度・湿度の計算]
    
    F --> G[2-G. 吸込温度・湿度の計算]
    
    G --> H[2-H. 未処理負荷の計算]
    
    H --> I[2-I. 集約・返却]
    
    I --> End([返却])
```

### calc_Q_UT_A の主要変数

| 変数名 | 単位 | 説明 | 次元 |
|--------|------|------|------|
| `Theta_ex_d_t` | ℃ | 外気温度 | 8760時間 |
| `X_ex_d_t` | kg/kg(DA) | 外気絶対湿度 | 8760時間 |
| `Theta_uf_d_t` | ℃ | 床下温度 | 8760時間 |
| `Theta_req_d_t_i` | ℃ | 要求室温 | 5区画×8760時間 |
| `X_req_d_t_i` | kg/kg(DA) | 要求絶対湿度（冷房のみ） | 5区画×8760時間 |
| `V_supply_d_t_i` | m³/h | 各区画への供給風量 | 5区画×8760時間 |
| `Theta_supply_d_t_i` | ℃ | 給気温度 | 5区画×8760時間 |
| `X_supply_d_t_i` | kg/kg(DA) | 給気絶対湿度（冷房のみ） | 5区画×8760時間 |
| `Theta_hs_out_d_t` | ℃ | 吹出温度 | 8760時間 |
| `Theta_hs_in_d_t` | ℃ | 吸込温度 | 8760時間 |
| `X_hs_out_d_t` | kg/kg(DA) | 吹出絶対湿度（冷房のみ） | 8760時間 |
| `X_hs_in_d_t` | kg/kg(DA) | 吸込絶対湿度（冷房のみ） | 8760時間 |
| `V_hs_supply_d_t` | m³/h | 給気風量合計 | 8760時間 |
| `V_hs_vent_d_t` | m³/h | 換気風量 | 8760時間 |


### 2-A. 初期化・気象データ読込

```mermaid

```

### 2-C. 各区画の設定温度・湿度の決定

```mermaid
flowchart TD
```

### 2-D. 各区画の供給風量の計算

```mermaid
flowchart TD
    Start[供給風量計算] --> Input[入力:<br/>- L_d_t_i 負荷<br/>- Theta_req_d_t_i 設定温度<br/>- Theta_supply_d_t_i 給気温度<br/>- X_req_d_t_i 設定湿度 冷房のみ<br/>- X_supply_d_t_i 給気湿度 冷房のみ]
    
    Input --> Calc[calc_V_supply_d_t_i]
    
    Calc --> Formula[V_supply = L / c_p×ρ×ΔT<br/>ΔT = Theta_req - Theta_supply<br/>冷房は潜熱も考慮]
    
    Formula --> Limit[風量制約<br/>- 最小風量<br/>- 最大風量]
    
    Limit --> Result[V_supply_d_t_i<br/>5区画×8760時間]
    
    style Calc fill:#e1ffe1
```

### 2-E. 給気温度・湿度の計算

```mermaid
flowchart TD
    Start[給気温度・湿度計算] --> Mix[混合計算]
    
    Mix --> Outdoor[外気導入分<br/>V_vent × Theta_ex<br/>V_vent × X_ex]
    Mix --> Return[還気分<br/>V_return × Theta_return<br/>V_return × X_return]
    
    Outdoor --> Combined[給気温度<br/>Theta_supply_d_t_i<br/>給気絶対湿度<br/>X_supply_d_t_i 冷房のみ]

    style Mix fill:#ffe1f5
```

### 2-F. 吹出温度・湿度の計算

```mermaid
flowchart TD
    Start[吹出温度・湿度計算] --> Agg[各区画への配分を集約]
    
    Agg --> Weight[風量加重平均<br/>Σ V_supply_i × Theta_supply_i<br/>/ Σ V_supply_i<br/>冷房は湿度も同様]
    
    Weight --> Result[Theta_hs_out_d_t<br/>X_hs_out_d_t 冷房のみ]

```

### 2-G. 吸込温度・湿度の計算

```mermaid
flowchart TD
    Start[吸込温度・湿度計算] --> Return[各区画からの還気を集約]
    
    Return --> Weight[風量加重平均<br/>Σ V_return_i × Theta_return_i<br/>/ Σ V_return_i<br/>冷房は湿度も同様]
    
    Weight --> Result[Theta_hs_in_d_t<br/>X_hs_in_d_t 冷房のみ]
    
    style Return fill:#e1ffff
```

### 2-H. 未処理負荷の計算

```mermaid
flowchart TD
    Start[未処理負荷計算] --> Capacity[機器処理能力の計算<br/>q_capacity = c_p × ρ × V × ΔT<br/>冷房は顕熱・潜熱分離]
    
    Capacity --> Compare{負荷 vs 能力}
    
    Compare -->|負荷 ≤ 能力| Zero[Q_UT_d_t_i = 0<br/>全負荷処理可能]
    Compare -->|負荷 > 能力| UnTreated[Q_UT_d_t_i =<br/>負荷 - 能力<br/>未処理負荷発生]
    
    Zero --> Result[未処理負荷<br/>5区画×8760時間]
    UnTreated --> Result
    
    style Compare fill:#ffffcc
    style UnTreated fill:#ffe1e1
```

```mermaid
flowchart LR

    R3 --> Return
    R4 --> Return
    R5 --> Return
    R6 --> Return
    
    style Sum fill:#e1e1e1
```

## ステップ4: 送風機消費電力の計算

```mermaid
flowchart TD
    Start[送風機消費電力] --> Type{計算モデル判定}
    
    Type -->|type1: ダクト式<br/>type2: RAC現行<br/>type4: 電中研| Standard[標準式<br/>get_E_E_fan_d_t]
    
    Type -->|type3: RAC潜熱| Latent[潜熱評価式<br/>latent_load.get_E_E_fan_d_t]
    
    Standard --> Formula[E_E_fan = f<br/>P_fan_rtd,<br/>V_hs_vent,<br/>V_hs_supply,<br/>V_hs_dsgn,<br/>q_hs,<br/>f_SFP]
    
    Latent --> FormulaL[E_E_fan = f<br/>V_hs_vent,<br/>q_hs,<br/>f_SFP]
    
    Formula --> Result[E_E_fan_d_t]
    FormulaL --> Result
    
    style Type fill:#ffffcc
```

**標準式の概要:**
```
E_E_fan_d_t = (V_hs_supply_d_t / V_hs_dsgn)^3 × P_fan_rtd / 1000
            または
              V_hs_vent_d_t × f_SFP / 1000
```
（詳細は条件分岐あり）
